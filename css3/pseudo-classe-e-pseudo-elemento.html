Pseudo Classe e Pseudo Elemento

Pseudo Classe:

Definição: é uma classe CSS pré-definida.

Existe duas classes pré-definidas para formatar o primeiro e o último elementos de uma lista.
São elas: first-child e last-child.
Vamos aplicar a formatação no primeiro e no último elementos na seguinte lista:

  • Domingo
  • Segunda-Feira
  • Terça-Feira
  • Quarta-Feira
  • Quinta-Feira
  • Sexta-Feira
  • Sábado

Vimos que ficou muito fácil formatar o primeiro e o último elementos da lista usando o first-child e o last-child.

Há uma outra pseudo-classe para formatar elementos de uma lista, a nth-child. Com ela, é possível formatar não apenas o primeiro e o último elementos, mas todos passando o índice do elemento.
O nth-child pode receber expressões aritméticas por parâmetro.
Vamos formatar os elementos pares e os elementos ímpares da seguinte lista:

  • Domingo
  • Segunda-Feira
  • Terça-Feira
  • Quarta-Feira
  • Quinta-Feira
  • Sexta-Feira
  • Sábado

Agora, vamos formatar as linhas de um elemento passando o índice.

  • Domingo
  • Segunda-Feira
  • Terça-Feira
  • Quarta-Feira
  • Quinta-Feira
  • Sexta-Feira
  • Sábado

Pseudo Classe de estado:

Definição: É uma pseudo classe que representa estados de elemento, ou seja, o que ele vai fazer se ocorrer determinado evento do usuário.

  • Hover : ´É uma pseudo-classe de estado que seleciona o link no momento em que passamos o mouse por cima dele.

Pseudo Elemento:

Definição: São elementos que não existem no documento mas podem ser selecionados pelo CSS.

O first-letter é um pseudo-elemento que altera apenas a primeira letra de um conteúdo.
