Superglobais

Muitas variáveis predefinidas em PHP são "superglobais", o que significa que estão sempre acessíveis, independentemente do escopo. Você pode acessá-los a partir de qualquer função, classe ou arquivo sem ter que fazer nada de especial.

As variáveis superglobais disponíveis em PHP são as seguintes:

$GLOBALS
 $_SERVER
 $_REQUEST
 $_POST
 $_GET
 $_FILES
 $_ENV
 $_COOKIE
 $_SESSION

Agora iremos falar sobre alguams delas!

PHP $GLOBALS

$GLOBALS é uma variável superglobal do PHP que é usada para acessar variáveis globais de qualquer lugar no script PHP (também dentro de funções ou métodos). O PHP armazena todas as variáveis globais em uma matriz chamada $GLOBALS[index]. O index contém o nome da variável. O exemplo abaixo mostra como usar a variável superglobal $GLOBALS:

<?php
$x = 75;
$y = 25;
  function addition() {
    $GLOBALS['z'] = $GLOBALS['x'] + $GLOBALS['y'];
}
  addition();
echo $z;
?>

PHP $_SERVER

$_SERVER é uma variável superglobal do PHP que contém informações sobre cabeçalhos, caminhos e locais de script. O exemplo abaixo mostra como usar alguns dos elementos em $_SERVER:

<?php
echo $_SERVER['PHP_SELF'];
echo <br>";
echo $_SERVER['SERVER_NAME'];
echo "<br>";
echo $_SERVER['HTTP_HOST'];
echo <br>";
echo $_SERVER['HTTP_REFERER'];
echo <br>";
echo $_SERVER['HTTP_USER_AGENT'];
echo <br>";
echo $_SERVER['SCRIPT_NAME'];
?>

Existem alguns elemntos que são mais importantes que podem entrar dentro do $_SERVER, você os encontra aqui.

PHP $_REQUEST

PHP $_REQUEST é usado para coletar dados depois de enviar um formulário HTML. O exemplo abaixo mostra um formulário com um campo de entrada e um botão de envio. Quando um usuário envia os dados clicando em "Enviar", os dados do formulário são enviados para o arquivo especificado no atributo de ação da marca <form>. Neste exemplo, apontamos este arquivo para o processamento de dados do formulário. Se você deseja usar outro arquivo PHP para processar os dados do formulário, substitua-o pelo nome do arquivo escolhido. Então, podemos usar a variável superglobal $_REQUEST para coletar o valor do campo de entrada:

<html>
<body>
<form method="post" action="">
  Name: <input type="text" name="fname">
  <input type="submit">
</form>
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    // collect value of input field
    $name = $_REQUEST['fname'];
    if (empty($name)) {
        echo "Name is empty";
    } else {
        echo $name;
    }
}
?>
</body>
</html>

PHP $_POST

PHP $_POST é amplamente utilizado para coletar dados do formulário depois de enviar um formulário HTML com method="post". $_POST também é amplamente utilizado para passar variáveis. O exemplo abaixo mostra um formulário com um campo de entrada e um botão de envio. Quando um usuário envia os dados clicando em "Enviar", os dados do formulário são enviados para o arquivo especificado no atributo de ação da marca <form>. Neste exemplo, apontamos para o próprio arquivo para processar os dados do formulário. Se você deseja usar outro arquivo PHP para processar os dados do formulário, substitua-o pelo nome do arquivo escolhido. Então, podemos usar a variável superglobal $_POST para coletar o valor do campo de entrada:

<html>
<body>
<form method="post" action="">
    Name: <input type="text" name="fname">
  <input type="submit">
</form>
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
        // collect value of input field
    $name = $_POST['fname'];
    if (empty($name)) {
        echo "Name is empty";
    } else {
        echo $name;
    }
}
?>
</body>
</html>

PHP $_GET

PHP $_GET também pode ser usado para coletar dados do formulário depois de enviar um formulário HTML com method = "get".
$_GET também pode coletar dados enviados no URL. Suponha que temos uma página HTML que contenha um hiperlink com parâmetros:

<html>
<body>
<a href="test_get.php?subject=PHP&web=W3schools.com">Test $GET</a>
</body>
</html>

Quando um usuário clica no link "Testar $ GET", os parâmetros "assunto" e "web" são enviados para "test_get.php" e você pode acessar seus valores em "test_get.php" com $_GET. O exemplo abaixo mostra o código em "test_get.php":

<html>
<body>
<?php
echo "Study " . $_GET['subject'] . " at " . $_GET['web'];{
?>
</body>
</html>
